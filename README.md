## Installation and usage

Clone this repository :

```
git clone git@github.com:GuilhermeGabriel/Calculator-ReactNative.git
cd Calculator-ReactNative
```

Install packages :

```
yarn install
```

When installation is complete, run with expo :

```bash
expo start
